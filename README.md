# Grace Glow English Academy

Grace Glow English Academy offers spoken English, grammar, IELTS preparation and interview coaching. Join our personalized English learning programs today.

## Project info

This project is a React application built with Vite, TypeScript, and Tailwind CSS.

## Getting Started

To get started with this project, follow these steps:

1.  Clone the repository.
2.  Install dependencies:
    ```sh
    npm install
    ```
3.  Start the development server:
    ```sh
    npm run dev
    ```

## Features

- Responsive design using Tailwind CSS
- Modern UI components using shadcn/ui
- Routing with React Router
- Form handling with React Hook Form

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
